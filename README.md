# DawnCraft 한국어 번역 프로젝트 (v2.0.16_hf)

본 프로젝트는 마인크래프트 모드팩 **DawnCraft - Echoes of Legends**의 한국어 번역 파일입니다. 가독성과 게임 플레이의 몰입감을 위해 지속적으로 업데이트 중입니다.

---

## 📢 제보 및 피드백
문법 오류, 미번역 텍스트, 혹은 문맥이 어색한 부분을 발견하시면 언제든지 **Issues** 섹션을 통해 제보해 주시기 바랍니다. 여러분의 피드백이 더 나은 번역을 만듭니다.

## 🛠 설치 방법

1. **파일 다운로드:** 원하는 버전의 번역 파일을 다운로드한 후 압축을 해제합니다.
2. **폴더 복사:** 해제된 폴더 내의 `Config`와 `global_packs` 폴더를 복사합니다.
3. **경로 붙여넣기:** 모드팩이 설치된 **루트(최상위) 폴더**에 붙여넣습니다.
   * 반드시 **'대상 폴더의 파일 덮어쓰기'**를 선택해야 합니다.
4. **언어 설정:** 게임 실행 후 [설정] > [언어] 메뉴에서 **한국어 (대한민국)**로 설정되어 있는지 확인합니다.

### 📂 CurseForge 기본 설치 경로 예시
> `C:\Users\사용자이름\curseforge\minecraft\Instances\DawnCraft - Echoes of Legends`

---

## ⚠️ 중요 사항
* 일부 번역이 정상적으로 출력되려면 게임 내 언어 설정이 반드시 **한국어 (대한민국)**로 되어 있어야 합니다.
* 모드팩 버전이 업데이트됨에 따라 번역 파일이 호환되지 않을 수 있으니 버전을 꼭 확인해 주세요.
